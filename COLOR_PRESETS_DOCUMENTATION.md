## 🎨 Create Your Own Palette

Don't like these presets? **Create your own!**

### Step-by-Step:
1. Go to **Shopify Admin → Online Store → Themes → Customize**
2. Click **Theme Settings** in the left sidebar
3. Click **Colors** section
4. Click any color swatch to open color picker
5. Either:
   - Enter a hex code directly (e.g., `#FF5733`)
   - Use the visual color picker
   - Click the eyedropper to pick from your screen

### Color Settings Explained:

- **Background** - Main page background
- **Surface** - Cards, sections background
- **Text** - Primary text color
- **Muted Text** - Secondary/supporting text
- **Accent** - Links, highlights, focus states
- **Rule/Border** - Lines, dividers, borders
- **Dark Background** - For dark sections/footer
- **Dark Text** - Text on dark backgrounds
- **Heading Color** - All heading text
- **Body Text Color** - Paragraph text
- **Nav Text** - Navigation menu text
- **Button Background** - Primary button color
- **Button Text** - Text on buttons

## 💡 Pro Tips

### Contrast is Key
- Ensure text colors have sufficient contrast with backgrounds
- Test your colors with [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
- Aim for at least 4.5:1 ratio for body text
- Aim for at least 3:1 ratio for large text/headings

### Color Psychology
- **Warm tones** (browns, tans, terracotta) = Natural, handmade, organic
- **Cool tones** (grays, sage) = Modern, minimal, calm
- **Dark themes** = Luxury, sophistication, elegance
- **Light themes** = Airy, approachable, friendly

### Brand Consistency
Use colors from your:
- Logo
- Product photography
- Packaging
- Brand guidelines

### Save Before Experimenting
Before changing colors:
1. Take screenshots of current settings
2. Or write down current hex codes
3. This makes it easy to revert if needed
