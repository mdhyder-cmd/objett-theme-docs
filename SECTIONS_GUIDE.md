# 📐 Section Recommendations - Quick Reference

**Updated comprehensive guide for all page types**

---

## 🏠 Homepage

**Recommended: 5 sections in this order**

1. **Hero** (Full-screen)
   - High-quality lifestyle image
   - 2-5 word heading
   - Action CTA

2. **Featured Product**
   - Best-selling item
   - Rich description
   - "View Product" CTA

3. **Material Story**
   - Craftsmanship photo
   - 3-4 sentences about quality
   - Builds trust

4. **New Arrivals**
   - 4-6 latest products
   - Grid layout
   - "View All" link

5. **Maker Story**
   - Founder photo
   - Personal story
   - "Learn More" CTA

**Optional sections:**
- Room Scene (if you have styled photography)
- Press Strip (if you have media mentions)

---

## 🛍️ Product Pages

**Standard Layout: 3 sections**

1. **Product Main** (REQUIRED)
   - Images (3-6 photos)
   - Title, price, variants
   - Buy button
   - Accordion details

2. **Product Material** (OPTIONAL)
   - Enable for artisan products
   - Craftsmanship details
   - Material attributes

3. **Product Recommendations** (REQUIRED)
   - Related products
   - Increases cart value
   - Auto-generated

**Templates available:**
- `product.json` - Standard items
- `product.made-to-order.json` - Custom orders
- `product.bundle.json` - Product sets

---

## 📚 Collection Pages

**Two template options:**

### Standard (Recommended for most)
- Clean product grid
- Filtering + sorting
- Fast loading
- **Use for:** Regular product browsing

### Editorial (Special collections)
- Hero banner
- Storytelling sections
- Richer layout
- **Use for:** Seasonal launches only

**Settings:**
- Products per page: 12 (optimal)
- Enable filters: Yes
- Enable sorting: Yes

## 📄 Content Pages

1. About Hero
2. About Story #1 (Origin)
3. About Values (3-4 principles)

## ✅ Best Practices

### Do:
✅ Use high-quality, authentic images  
✅ Keep homepage to 4-5 sections  
✅ Write concise, benefit-focused copy  
✅ Test on mobile devices  
✅ Use consistent image styles  

### Don't:
❌ Overcrowd pages with sections  
❌ Use stock photos (use real photos)  
❌ Write long paragraphs (keep scannable)  
❌ Hide key info in accordions  
❌ Forget to optimize for mobile

## 📱 Mobile Considerations

All sections are mobile-optimized:
- Hero: Auto-adjusts text position
- Grids: Stack to single column
- Images: Responsive sizing
- Text: Readable font sizes

Always preview on mobile before publishing!
