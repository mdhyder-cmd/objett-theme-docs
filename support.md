## How to Get Help

### What to Include in Your Support Request

To help us assist you faster, please include:

1. **Your Shopify Store URL**
   - Example: `yourstore.myshopify.com`

2. **Detailed Description**
   - What are you trying to do?
   - What's happening instead?
   - When did the issue start?

3. **Screenshots or Screen Recording**
   - Visual context helps us understand the issue
   - Use tools like: Loom, CloudApp, or Windows Snipping Tool

4. **Steps to Reproduce**
   - What exact steps did you take?
   - Can you recreate the issue consistently?

5. **Browser & Device Information**
   - Which browser? (Chrome, Safari, Firefox, etc.)
   - Desktop or mobile?
   - Any browser extensions enabled?

### Email Template

```
Subject: Maison Theme Support - [Brief Issue Description]

Store URL: yourstore.myshopify.com
Issue: [Detailed description of your problem]

Steps I've taken:
1. [What you tried]
2. [Other attempts]

Screenshots: [Attach images]

Browser: Chrome (or Safari, Firefox, etc.)
Device: Desktop (or Mobile, Tablet)
```

## Frequently Asked Questions

### Installation & Setup

**Q: How do I install the theme?**  
A: Download the ZIP file from Shopify Theme Store, then upload it via **Online Store > Themes > Upload theme**. See [Installation Guide](documentation#installation).

**Q: Can I preview the theme before publishing?**  
A: Yes! After uploading, click **Customize** to preview and edit before clicking **Publish**.

**Q: How do I switch between style presets?**  
A: In the theme editor, go to **Theme Settings**, click the **Presets** dropdown at the top, and select your preferred style.

---

### Customization

**Q: How do I change colors?**  
A: Go to **Theme Settings > Colors** and customize each color option. See [Color Settings](documentation#color-settings).

**Q: How do I change fonts?**  
A: Go to **Theme Settings > Typography** and select fonts from Shopify's font library. See [Typography Settings](documentation#typography-settings).

**Q: Can I add my own CSS?**  
A: Yes, advanced users can add custom CSS through the theme editor or by editing theme files.

**Q: How do I change the homepage layout?**  
A: In the theme editor, add/remove/reorder sections. See [Homepage Setup](documentation#homepage-setup).

---

### Features

**Q: Is the theme mobile responsive?**  
A: Yes! The theme is fully responsive and tested on all devices and screen sizes.

**Q: Does it work with multiple currencies?**  
A: Yes, the theme supports Shopify's multi-currency feature. Enable it in your Shopify settings.

**Q: Can customers save items to a wishlist?**  
A: Yes, the wishlist feature saves items to the browser (no login required).

**Q: How does the cart drawer work?**  
A: When customers add items to cart, a drawer slides out from the right showing their cart without leaving the page.

**Q: Can I use video backgrounds?**  
A: Yes, the hero section supports both YouTube and Vimeo video backgrounds.

---

### Troubleshooting

**Q: Images aren't showing up**  
A: Check file formats (JPG, PNG, WebP), file size (under 20MB), and re-upload if needed.

**Q: Changes aren't saving**  
A: Make sure you click the **Save** button in the top right of the theme editor.

**Q: Theme looks different than preview**  
A: Clear your browser cache and hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac).

**Q: Sections are missing**  
A: Check if sections are hidden (eye icon) or if you're editing the correct theme template.

---

### Technical

**Q: What browsers are supported?**  
A: All modern browsers: Chrome, Firefox, Safari, Edge (latest versions).

**Q: Is the theme accessible?**  
A: Yes, the theme follows WCAG accessibility guidelines and includes proper ARIA labels.

**Q: Is the theme SEO optimized?**  
A: Yes, includes semantic HTML, structured data (JSON-LD), and proper meta tags.

**Q: Does it work with Shopify 2.0?**  
A: Yes, the theme is built on Shopify's latest theme architecture (Online Store 2.0).
